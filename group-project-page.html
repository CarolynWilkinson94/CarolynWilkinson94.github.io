She Codes Full Stack Group Project

This project was completed as the final capstone task for the She Codes program, where our team designed and built a full-stack web application from concept through to delivery. My primary contributions were on the backend using Django and REST APIs, where I helped implement core functionality, supporting logic, and API integration. I also played a key role in project management, assisting with planning, task coordination, and keeping the team aligned on scope, priorities, and deadlines. In addition, I contributed to frontend bug fixes and final UI refinements, ensuring the application was stable, polished, and ready for delivery. The project involved collaborative planning, version control with Git, and regular communication within the team. It reflects my ability to contribute reliably in a team environment, manage tasks effectively, and support a project through to completion.
