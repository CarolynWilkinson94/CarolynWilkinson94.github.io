She Codes Plus Website Project

In this project, we were tasked to recreate a She Codes demo website in order to learn and practice HTML and CSS skills. We began by using the demo site as our wireframe and building the HTML structure of the site from that. We learend about creating divs, sections, articles, headers and all other basic structural elements of a HTML website. We then added CSS to style the website to look like the demo, which involved learning how apply the many different classes and elements of CSS.
