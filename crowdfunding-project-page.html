She Codes Crowdfunding Project

This was my first solo full-stack project, built with Django and React as part of the She Codes Plus program and developed across both the Django and React units. I designed and implemented the backend first using Django, followed by building the frontend in React, connecting the two through API endpoints. The project is based on a crowdfunding-style platform that connects underprivileged students with volunteer teachers or musicians. Students can post projects they need support with, while educators pledge their time to meet online and provide guidance. One of the main challenges was designing the backend architecture before the frontend existed, which strengthened my ability to plan data models, APIs, and application flow in advance.
