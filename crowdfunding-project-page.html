She Codes Crowdfunding Project

This is my first solo full stack project, built using Django and React. I built this as part of the She Codes Plus Program, and built it across the Django and React units. I built the backend first using Django and then the frontend using React. I chose to use the concept of a crowdfunding site that would connect underprivileged students with volunteer teachers or musicians. Students would post projects they would like help with, and musicians/teachers would pledge their time to meet online and help with the projects. The biggest challenge in this project was trying to visualise how my backend would connect to my frontend without seeing the frontend.
