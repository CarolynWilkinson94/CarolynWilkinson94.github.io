She Codes Plus Website Project

This project involved recreating a She Codes demo website to build and practise core HTML and CSS skills. Using the demo site as a wireframe, I structured the page with semantic HTML elements such as sections, articles, headers, and divs to create a clean, accessible layout. I then applied CSS to replicate the design of the original site, learning how to use classes, selectors, and styling rules to control layout and visual hierarchy. This project strengthened my understanding of front-end fundamentals and translating a design into functional code.
