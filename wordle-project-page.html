TAFE Python Wordle CLI Project

I developed this terminal-based Wordle-style game in Python as part of my Certificate IV in Information Technology (Web Development) at TAFE. The project was designed to strengthen my understanding of core programming concepts, including control flow, functions, data structures, and file handling. The game validates user input against a dictionary, scores guesses using Wordle-style feedback, and tracks attempts across six rounds. Game results are written to a file for persistence. I also wrote simple test cases to verify the scoring logic. While the core application runs in the terminal, I created a visual UI mock for my portfolio to clearly demonstrate the gameplay flow and scoring logic in a more accessible, front-end-friendly format. Tech: Python
